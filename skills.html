Skills

Skills are something that your character can put points into. The higher the amount of points, the easier it becomes to do a check for that skill. A check is like an attack roll- it takes into account training (skill rank), natural talent (ability modifier), and luck (the die roll).

For a list of skills, click here.

Skill Check Roll:
1d20+ skill rank + ability modifier(the skill stat) + misc. modifiers

Levelling Skills

Different classes get different amounts of skill points every level. If you buy a class skill, your character gets 1 rank (equal to a +1 bonus on checks with that skill) for each skill point. If you buy other class skills (cross-class skills), you get .5 rank per skill point.

Your maximum rank in a class skill is your character level + 3.

Your maximum rank in a cross-class skill is one-half of this number (do not round up or down).

Using Skills

Some checks are made against a Difficulty Class. The Difficulty Class is a number that you must score equal to or above in order to succeed.

Table: Difficulty Class Examples
Difficulty (DC) Example (Skill Used)
Very easy (0) Notice something large in plain sight (Spot)
Easy (5) Climb a knotted rope (Climb)
Average (10) Hear an approaching guard (Listen)
Tough (15) Rig a wagon wheel to fall off (Disable Device)
Challenging (20) Swim in stormy water (Swim)
Formidable (25) Open an average lock (Open Lock)
Heroic (30) Leap across a 30-foot chasm (Jump)
Nearly impossible (40) Track a squad of centaurs across hard ground after 24 hours of rainfall (Survival)
